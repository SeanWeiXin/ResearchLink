# Excel 上传和图表功能使用指南

## 🎉 新增功能

ResearchLink 现已支持 Excel 数据上传、处理和可视化图表创建！

## 📋 功能列表

### 1. Excel 文件上传
- 📤 支持拖拽上传
- 📊 支持 .xlsx, .xls, .csv 格式
- 📏 最大 50MB 文件大小
- ⚡ 自动数据分析和识别

### 2. 数据预览与配置
- 👁️ 数据预览（前 10 行）
- 📑 工作表选择（多工作表文件）
- 📅 日期列识别
- 🔢 数值列识别
- ⚙️ 自定义处理配置

### 3. 图表创建器
- 📈 5 种图表类型：
  - 折线图（时间序列数据）
  - 柱状图（数据比较）
  - 面积图（变化趋势）
  - 散点图（数据分布）
  - 饼图（占比分析）
- 🎨 自定义颜色方案
- 📝 标题和描述
- 🔍 数据列映射

### 4. 图表展示与管理
- 📊 ECharts 专业渲染
- ✏️ 编辑图表信息
- 🔗 发布到论坛帖子
- 👀 浏览量统计
- 🔒 公开/私有设置

## 🚀 使用流程

### 步骤 1：获取上传权限

默认情况下，新用户没有上传权限。需要管理员授权：

1. 访问 **权限管理** 页面（http://localhost:5173/quick-admin）
2. 找到自己的用户账户
3. 点击"编辑"
4. 开启"上传 Excel"权限
5. 保存

或者联系管理员为您授权。

### 步骤 2：上传 Excel 文件

1. 点击顶部导航栏的 **"上传 Excel"** 按钮
2. 拖拽文件或点击选择文件
3. 等待上传完成
4. 自动跳转到数据预览页面

**支持的数据格式：**
- 时间序列数据（日期 + 数值）
- 横截面数据
- 面板数据

### 步骤 3：配置数据

1. **选择工作表**（如果有多个）
2. **选择日期列** - 用于 X 轴的时间数据
3. **选择数值列** - 要分析的数据列（可多选）
4. 点击 **"保存配置"**

配置完成后，系统会显示"配置完成"提示。

### 步骤 4：创建图表

1. 点击 **"创建图表"** 按钮
2. **选择图表类型**：
   - 折线图：适合时间序列
   - 柱状图：适合类别比较
   - 面积图：强调变化趋势
   - 散点图：展示分布
   - 饼图：显示占比
3. **填写基本信息**：
   - 图表标题
   - 描述（可选）
4. **配置数据映射**：
   - X 轴数据列
   - Y 轴数据列
5. **选择颜色方案**（可选）
6. 点击 **"创建图表"**

### 步骤 5：查看和分享

1. 自动跳转到图表详情页
2. 查看渲染好的图表
3. 可以：
   - 编辑图表信息
   - 发布到论坛帖子
   - 全屏查看
   - 删除图表

## 💡 最佳实践

### 数据准备

**推荐的 Excel 格式：**

| date       | value1 | value2 | value3 |
|------------|--------|--------|--------|
| 2024-01-01 | 100    | 50     | 75     |
| 2024-01-02 | 105    | 52     | 78     |
| 2024-01-03 | 103    | 51     | 76     |

**日期格式支持：**
- YYYY-MM-DD (2024-01-01)
- YYYY/MM/DD (2024/01/01)
- DD-MM-YYYY (01-01-2024)
- ISO 8601 (2024-01-01T10:00:00)

### 图表选择建议

| 数据类型 | 推荐图表 | 说明 |
|---------|---------|------|
| 时间序列 | 折线图/面积图 | 展示趋势变化 |
| 类别比较 | 柱状图 | 对比不同类别 |
| 占比分析 | 饼图 | 显示部分与整体关系 |
| 相关性分析 | 散点图 | 展示变量间关系 |

### 颜色搭配

系统提供默认配色方案，也可以自定义：
- 蓝色系：专业、科技感
- 绿色系：环保、增长
- 暖色系：活力、突出
- 单色系：简洁、统一

## 🔧 技术说明

### 前端技术栈
- React 18
- Ant Design 组件库
- ECharts 图表库
- echarts-for-react React 封装

### 后端 API

**上传接口：**
- `POST /api/uploads/excel` - 上传文件
- `GET /api/uploads/my-uploads` - 获取列表
- `GET /api/uploads/:id` - 获取详情
- `PUT /api/uploads/:id/config` - 更新配置
- `DELETE /api/uploads/:id` - 删除

**图表接口：**
- `POST /api/charts` - 创建图表
- `GET /api/charts` - 获取列表
- `GET /api/charts/:id` - 获取详情
- `PUT /api/charts/:id` - 更新图表
- `DELETE /api/charts/:id` - 删除图表
- `POST /api/charts/:id/publish` - 发布到帖子

### 数据处理流程

```
上传 Excel 
  ↓
解析文件（后端）
  ↓
分析数据结构
  ↓
识别日期/数值列
  ↓
保存元数据
  ↓
配置处理参数
  ↓
创建图表配置
  ↓
生成 ECharts 选项
  ↓
渲染图表
```

## ❓ 常见问题

### Q1: 上传失败？

**A:** 检查以下几点：
1. 是否有上传权限
2. 文件格式是否正确
3. 文件大小是否超过 50MB
4. 网络连接是否正常

### Q2: 数据预览为空？

**A:** 可能的原因：
1. 文件还在处理中
2. 数据格式无法识别
3. 需要手动配置列类型

### Q3: 图表创建失败？

**A:** 确保：
1. 已保存数据配置
2. 选择了必需的列
3. 数据格式正确

### Q4: 如何修改已创建的图表？

**A:** 
1. 访问图表详情页
2. 点击"编辑"按钮
3. 修改标题、描述等
4. 保存即可

### Q5: 可以重新配置数据吗？

**A:** 
1. 访问上传预览页
2. 修改配置
3. 重新保存
4. 基于新配置创建新图表
