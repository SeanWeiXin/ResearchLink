## 🆘 常见问题

### Q1: Render 验证失败怎么办？
**A**: 确保信用卡支持国际支付。可以尝试：
- Visa/Mastercard 信用卡
- 某些借记卡（支持外币）
- 虚拟信用卡（如一些在线服务）

### Q2: 如果 Render 一直验证不通过怎么办？
**A**: 使用替代方案：
- **方案 A**: 使用 Vercel 部署后端（需要调整代码）
- **方案 B**: 使用 Railway（同样需要信用卡，但通过率更高）
- **方案 C**: 使用 Fly.io（也有免费额度）

### Q3: 部署后出现 CORS 错误怎么办？
**A**: 
1. 检查 Render 的 CORS_ORIGIN 环境变量是否正确
2. 确保格式为 `https://your-domain.vercel.app`（不要带斜杠）
3. 重新部署后端服务

### Q4: 数据库连接失败怎么办？
**A**:
1. 检查 MongoDB Atlas 的网络访问配置（0.0.0.0/0）
2. 验证连接字符串是否正确
3. 确认数据库用户名密码是否正确
4. 在 MongoDB Atlas 查看集群状态
